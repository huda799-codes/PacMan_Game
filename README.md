This project is a simple implementation of the classic Pac-Man game developed in Java as part of an Object-Oriented Programming (OOP) lab assignment. The main objective of this project was to apply core OOP concepts like classes, objects, encapsulation, and modular design in a practical way.

The game allows the player to control Pac-Man using keyboard inputs, collect food items to increase score, and avoid ghosts. A special feature called frightened mode is triggered when Pac-Man eats a cherry, allowing him to temporarily defeat ghosts.

The game is built using Java Swing for the graphical interface and uses a timer-based game loop to handle updates and rendering.

Features:
Grid-based movement system
Score and lives tracking
Multiple ghosts with basic behavior
Frightened mode (power-up system)
Keyboard controls (Start, Pause, Reset)
Image-based rendering for characters and map
Technologies Used
Java
Java Swing (GUI)
IntelliJ IDEA

Controls:
S → Start game
P → Pause game
R → Reset game
Arrow Keys → Move Pac-Man

Purpose:
This project was created for learning and practicing OOP concepts and understanding how simple game logic can be implemented using Java.
